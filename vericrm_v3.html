This CRM reads your Google Sheet directly using the Sheets API.
No Apps Script. No server. Just paste 2 values below and you're live.
Step 1 — Google Sheet ID
Open your sheet. The URL looks like:
docs.google.com/spreadsheets/d/THIS_PART/edit
Copy that middle part and paste below.
Step 2 — Apps Script Web App URL (for saving entries)
1. Open your Google Sheet → Extensions → Apps Script
2. Paste the VeriCRM_AppScript.gs code → Save (💾)
3. Click Deploy → New Deployment → Web App
4. Execute as: Me · Who has access: Anyone
5. Click Deploy → Copy the Web App URL
Step 2b — Google API Key (for reading sheet directly — optional if using Apps Script)
Only needed as fallback. Leave blank if Apps Script URL is set above.
Set sheet to "Anyone with link can view" if using API key.
Step 3 — Sheet Tab Name
The tab at the bottom of your sheet that has the leads (from your screenshot: ALL_USERS)
Step 4 — Call Logs Tab Name
A separate tab where call entries will be saved. Create an empty tab in your sheet named this:
